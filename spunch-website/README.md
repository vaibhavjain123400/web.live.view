# Spunch Website

Welcome to the Spunch website project, developed by Bhagwati Steels. This project showcases a modern and professional web presence for our premium sanitary ware brand.

## Project Structure

The project is organized as follows:

```
spunch-website
├── src
│   ├── index.html          # Main HTML file
│   ├── css
│   │   └── styles.css      # CSS styles for the website
│   ├── js
│   │   └── main.js         # JavaScript for interactive elements
│   ├── components
│   │   ├── header.html     # Header component
│   │   ├── hero.html       # Hero section component
│   │   ├── about.html      # About section component
│   │   ├── products.html   # Products section component
│   │   ├── price-list.html  # Price list section component
│   │   ├── contact.html     # Contact section component
│   │   └── footer.html      # Footer component
│   └── data
│       └── products.json    # Product data in JSON format
├── package.json             # npm configuration file
├── .gitignore               # Files to ignore in version control
└── README.md                # Project documentation
```

## Features

- **Responsive Design**: The website is designed to be fully responsive, ensuring a seamless experience on all devices.
- **Clean and Premium Look**: The design focuses on a clean aesthetic that reflects the quality of our products.
- **Interactive Elements**: JavaScript is used to enhance user interaction, including smooth scrolling and mobile menu toggling.

## Setup Instructions

1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd spunch-website
   ```
3. Open the `index.html` file in your web browser to view the website.
